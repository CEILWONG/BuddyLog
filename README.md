# BuddyLog

像老友一样聊天，像岁月一样记录。一款 AI 聊天日记 Agent 应用，帮助使用者无痛写日记。

## 核心特点

- **自然对话** - 像和老朋友聊天一样记录日常，无需刻意写日记
- **智能归档** - 自动将对话整理成优美的日记文章
- **长期记忆** - AI 记住你的喜好、经历和重要事项
- **时光追溯** - 侧边栏展示历史日记，随时回顾过往

## 技术架构

## 快速开始

### 环境要求

- Python 3.8+
- 阿里云 DashScope API Key

### 安装依赖

```bash
pip install -r requirements.txt
```

### 配置环境变量

创建 `.env` 文件：

```env
DASHSCOPE_API_KEY=your_api_key_here
MODEL_NAME=qwen-plus
```

### 启动服务

```bash
python main.py
```

访问 http://localhost:8000

## 核心功能

### 1. 智能聊天
- 基于用户档案和长期记忆生成个性化回复
- 支持多轮对话上下文
- 对话内容自动保存到草稿

### 2. 日记归档
- 点击"完成今日"手动归档
- 自动将过期草稿归档（非当天）
- 生成第一人称叙事风格的日记文章

### 3. 长期记忆
- 每次归档后自动更新记忆
- 基于原始对话 + 日期上下文融合新旧信息
- 用于后续聊天的个性化回复

### 4. 文件管理
- **草稿文件**: `diary_YYYY-MM-DD_draft.md` - 当日对话记录
- **归档文件**: `diary_YYYY-MM-DD_N.md` - 完成的日记
- **系统文件**: `profile.md`, `memory.md` - 用户画像和记忆

## 数据流

```
用户对话 → 草稿文件（实时保存）
    ↓
点击"完成今日" → AI提取结构化摘要
    ↓
AI生成日记文章（基于原始对话）
    ↓
更新长期记忆（基于原始对话+日期）
    ↓
保存归档文件
```

## API 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/` | GET | 前端页面 |
| `/chat` | POST | 聊天接口 |
| `/files` | GET | 获取文件列表 |
| `/file/{filename}` | GET | 读取文件内容 |
| `/archive` | POST | 手动归档 |

## 前端体验优化

- **输入动画** - 发送消息后显示 "..." 闪亮动画，提示正在准备回复
- **归档提示** - 点击"完成今日"显示"正在整理今日记录..."加载状态
- **实时反馈** - 每次对话后显示"Buddy 已记下~"提示

## 自定义配置

编辑 `data/profile.md`：

```markdown

你是 Buddy，一个温暖、幽默的老朋友...

## 用户档案
- 职业: 程序员
- 爱好: 跑步、阅读
```
